# BasicMinestomServer


## Features

* Commands:
  * ``stop``
  * ``gamemode``
  * ``fly``
  * ``flyspeed``
  * ``spawn``
  * ``teleport``
  * ``teleporthere``
* Async Log Files
* Colors in Console (ColorCode support)

## Requirements

These things are required for use:

* Java 17
* Minecraft 1.18.2
